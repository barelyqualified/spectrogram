# Spectrogram

![A screenshot of the spectrogram program.](example.png)

*A waveform and spectrogram of "[Formula]" from Aphex Twin's album* Windowlicker,
*showing a hidden face.*

This program can display waveforms and spectrograms for audio files, as well as the reverse process, converting spectrogram images back into audio.

# Fork info (09.07.26)
All I did was add the needed external libraries for the compilation.
I used java-11-openjdk with Eclipse IDE on Compiler compliance level 11 to compile it.
Download the release from the release page on the top right. 
## Running
Use any JRE which is compatible (JRE 11+). The JAR is self-contained with all libraries embedded.
```bash
java -jar Spectrogram.jar
```
